# MTCRE
This repository contains the command references, configuration examples, and lab scripts used throughout the MikroTik Certified Routing Engineer (MTCRE) training labs.  It is designed to support students and network engineers in understanding, replicating, and mastering advanced routing concepts using MikroTik RouterOS.
